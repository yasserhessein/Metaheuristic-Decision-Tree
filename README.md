# Water Quality Classification System
### Decision Tree Optimized with Metaheuristic Algorithms (ABC, GWO, TSO)

A comprehensive framework integrating SMOTE data balancing and metaheuristic optimization for accurate water quality classification, developed by **Dr. Yasir Hussein Shakir**.

## 📋 Overview

This project implements an integrated framework for water quality classification that combines **SMOTE** (Synthetic Minority Oversampling Technique) for handling class imbalance with three metaheuristic optimization algorithms (**Artificial Bee Colony - ABC**, **Grey Wolf Optimizer - GWO**, and **Tuna Swarm Optimization - TSO**) to automatically tune Decision Tree hyperparameters, achieving superior predictive performance on water potability datasets.

---

## 🚀 Features

* **SMOTE-based Data Balancing:** Effectively addresses severe class imbalance issues (88.6% vs 11.4%).
* **Metaheuristic Optimization:** Three advanced algorithms implemented for Decision Tree (DT) hyperparameter tuning:
    * 🐝 **Artificial Bee Colony (ABC)**
    * 🐺 **Grey Wolf Optimizer (GWO)**
    * 🐟 **Tuna Swarm Optimization (TSO)**
* **Comprehensive Evaluation Metrics:** Models are evaluated using Accuracy, Precision, Recall, F1-Score, and MCC.
* **GUI-Based Deployment:** Features an interactive Tkinter interface for real-time water quality prediction.
* **Result Logging:** Automatically logs prediction results to a text file with timestamps and confidence scores.
* **Visual Indicators:** Color-coded prediction outputs (**Green = Safe**, **Red = Not Safe**).
* **Robust Interface:** Built-in input validation and forms reset functionality.
* **High Efficiency:** Optimized for real-time deployment with a lightweight framework (< 50 ms prediction latency).
